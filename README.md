# 🔧 Fix It – by Dummies Build

A fast-paced browser repair game built alongside our MacBook Pro repair series. Tap bugs before the timer runs out, chain combos, survive five stages of escalating chaos — and try not to brick it.

## Features
- 5 repair stages with increasing difficulty (Power Issue → Final Boss)
- 3 difficulty modes — Easy, Normal, Hard (brutally short timers)
- Combo system with Fire Mode at 5x multiplier
- Panic bugs that dart around the screen
- Particle explosions, screen shake, combo flash animations
- Letter grade on results (S / A / B / C / D)
- Local high score persistence
- 4 device types — Laptop, Phone, Desktop, Watch
- Fully mobile and touch compatible
- Single file — no dependencies, no build step

## Stack
Vanilla HTML / CSS / JS — one file, zero frameworks.
